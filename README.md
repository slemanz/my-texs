# my-texs

A personal collection of generic LaTeX templates (`.tex`) and custom class files
(`.cls`) — built to be reused, adapted, and extended.

---

## Repository Structure

```
my-texs/
├── README.md
├── example/
│   ├── example.tex       # A generic LaTeX document to get you started
│   └── README.md         # This guide: setup + compilation on Ubuntu
└── ...                   # Your .tex and .cls files go here
```

---

## Getting Started

Head over to the [`example/`](./example/) folder to learn how to set up LaTeX on
Ubuntu and compile your first document from the terminal.
